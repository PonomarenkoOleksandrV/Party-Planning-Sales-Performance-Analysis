📊 Project: Party Planning & Sales Performance Analysis
Project Overview
The project consists of two parts: comprehensive analysis and planning of a large event (300+ guests) and analysis of the company's sales performance (Northwind) using SQL. The main goal is to transform raw data into understandable insights for cost optimization and performance evaluation.

What work did I do:
Infrastructure and ETL:

Deployed and configured Microsoft SQL Server.

Used DBeaver as the main client for writing queries and managing the database.

Imported and prepared data (Data Cleaning) for further analysis.

Data Analysis (SQL):

Wrote complex SQL queries to calculate key metrics: Total Sales, Average Order Value, Average Time to Ship.

Calculated specific indicators such as Top5Ratio (share of top 5 products in category sales) and Order Frequency.

Visualization (Tableau):

Created an interactive dashboard in Tableau.

Implemented complex Calculated Fields to segment guests by age and preferences.

Configured Filter Actions for easy navigation.

Data Sources:
Northwind database (MS SQL Server dialect).

Datasets for party planning (guest lists, addresses, cost estimates).
